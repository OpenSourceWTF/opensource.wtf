# Hello, World!

Welcome to **Opensource.wtf**, the home of our open source projects.

We build tools for developers who work with AI agents, automation, and modern workflows. Everything we make is open source because we believe the best software is built in the open.

## What We're Building

Our current projects span the AI agent ecosystem:

- **[Capybara](https://github.com/OpenSourceWTF/capybara)** — A framework for building AI agents that reason, plan, and execute
- **[WAAAH](https://github.com/OpenSourceWTF/WAAAH)** — Workflow automation designed specifically for AI agents
- **[Dojo](https://github.com/OpenSourceWTF/dojo)** — A skill registry so AI coding agents can learn and share capabilities

## Why "WTF"?

Because the best reaction to great open source is "wait, this is free?" We want to build things that make people say WTF in the best way possible.

Stay tuned for more posts about our projects, the tech behind them, and the open source philosophy that drives us.
